SLIDE 2: INTRODUCTION & PROBLEM STATEMENT
-------------------------------------------------------
The Problem:
- Traditional library management is manual and error-prone.
- Tracking borrow limits, due dates, and penalties is difficult.
- Students lack real-time visibility of book availability.

The Solution:
- A digital, automated platform for the CTC Library.
- Seamless interaction between Students and Administrators.
- Real-time catalog, automated calculations, and secure data management.
-------------------------------------------------------

SLIDE 3: SYSTEM OVERVIEW
-------------------------------------------------------
Core Modules:
1. Student Portal:
   - For browsing books, checking availability, and managing personal activity.
2. Admin Dashboard:
   - For managing the catalog, users, and approving/rejecting requests.
3. Backend Server:
   - Handles logic for borrowing, returns, penalties, and security.
-------------------------------------------------------

SLIDE 4: KEY FEATURES - STUDENT
-------------------------------------------------------
- Smart Catalog: Search books by title or author with "Available/Out of Stock" status.
- Instant Requests: Request to borrow books directly (1 click + duration input).
- Dashboard: "My Books" section to track:
  - Pending Requests
  - Issued Books (Current)
  - Return Deadlines
- User Experience: Modern interface with Toast notifications (no annoying popups).
-------------------------------------------------------

SLIDE 5: KEY FEATURES - ADMIN
-------------------------------------------------------
- Quick Actions Hub:
  1. Add/Remove Books
  2. Approve Borrow Requests
  3. Issue & Return Books
  4. Manage Students
- Automated Logic:
  - Auto-calculates Due Dates based on requested days.
  - Auto-calculates Penalty Fees (Rs 2/day) for late returns.
- Insights:
  - Real-time stats (Total Books, Borrowed, Overdue).
  - Transaction History log.
-------------------------------------------------------

SLIDE 6: TECHNOLOGY STACK
-------------------------------------------------------
"Built with the MERN Stack"

- Frontend (Client-Side):
  - React.js + Vite (Fast, responsive UI)
  - Tailwind CSS (Modern styling)
  - Lucide React (Icons)

- Backend (Server-Side):
  - Node.js & Express.js (RESTful API)
  - MongoDB & Mongoose (Database)
  - JWT (Secure Authentication)
-------------------------------------------------------

SLIDE 7: IMPROVEMENTS & INNOVATIONS
-------------------------------------------------------
What makes this project stand out?
1. Custom "Fixed Viewport" Layout: Header/Sidebar stay fixed while content scrolls.
2. Dynamic Penalty System: Automatically charges for late returns.
3. Secure Role-Based Access: Admins have full control but are restricted from borrowing (preventing conflict of interest).
4. Interactive UI: Replaced browser alerts with custom Modals and Toasts.
